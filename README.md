# 📊 Dashboard de Rentabilidad E-Commerce: Limpieza y Visualización

## 🌟 Descripción del Proyecto
Este proyecto forma parte de mi formación constante en la **Licenciatura en Ciencia de Datos**. El objetivo principal fue transformar un conjunto de datos con graves problemas de integridad en una herramienta de toma de decisiones estratégica utilizando **Power BI**.

## 🛠️ Desafíos Técnicos Superados
El mayor valor de este proyecto radica en la fase de **ETL**, donde se resolvieron los siguientes problemas de calidad de datos:

* **Corrección de Notación Científica**: Transformación de valores ilegibles (ej. `6E+18`) en campos numéricos precisos mediante Power Query.
* **Normalización de Porcentajes**: Resolución de errores de cálculo que mostraban márgenes superiores al 10,000%, logrando un margen real coherente del **30.04%**.
* **Eliminación de Errores de Origen**: Limpieza de registros corruptos que impedían el correcto modelado de los datos.

## 📈 Herramientas y Técnicas
* **Power Query**: Limpieza profunda, tipificación de datos y normalización.
* **DAX**: Creación de medidas para Ganancia Real, Margen Real % y KPIs dinámicos.
* **UI/UX Design**: Implementación de un diseño minimalista con jerarquía visual, bordes redondeados y sombras para mejorar la legibilidad.
